Financial Forecasting (Recursive Algorithm)

Project Files:
1. FinancialForecast.java

Concept:
- Uses recursion to calculate future value.
- Formula:
  Future Value = Present Value * (1 + Growth Rate)^Years

Time Complexity:
O(n)

Space Complexity:
O(n)

Compile:
javac FinancialForecast.java

Run:
java FinancialForecast
